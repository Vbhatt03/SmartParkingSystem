NOTES FOR TESTING
================================================================================

1. To test the system yourself:
   
   a) Compile:
      javac -d bin src/com/smartpark/exceptions/*.java \
                    src/com/smartpark/utils/*.java \
                    src/com/smartpark/users/*.java \
                    src/com/smartpark/io/*.java \
                    src/com/smartpark/*.java
   
   b) Run:
      java -cp bin com.smartpark.Main
   
   c) Use default credentials:
      Admin: admin / admin123
      Attendant: attendant / att123
      Customer: customer1 / cust123

2. After running, check data/ directory for persistent files:
   - users.txt
   - bookings.txt
   - payments.txt
   - system_logs.txt

3. Test data persistence:
   - Create bookings
   - Exit system
   - Restart system
   - Verify bookings still exist

4. Test exception handling:
   - Try to book with no slots
   - Try to cancel completed booking
   - Check error logs

5. Test file I/O:
   - Monitor data files being updated
   - Check log file growth
   - Verify format consistency
